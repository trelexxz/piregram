## 🧠 Overview

**Piregram** — это мессенджер в стиле Telegram, который работает через **Radmin VPN**.

Он не требует серверов или регистрации — только локальная сеть через VPN.

> 💡 Идеально для:
>
> * общения в закрытых сетях
> * LAN-проектов
> * обхода блокировок
> * приватных чатов

---

## 🚀 Features

* 💬 Чаты в реальном времени
* 🔒 Полностью приватная сеть
* ⚡ Работа без интернета (через Radmin VPN)
* 👥 Поддержка групповых чатов
* 🖥️ Лёгкий и быстрый клиент
* 🧩 Open-source и расширяемый

---

## ⚙️ How It Works

1. Установи **Radmin VPN**
2. Подключись к одной сети с друзьями
3. Запусти Piregram
4. Общайся напрямую через локальную VPN-сеть

```text
User A ──┐
         ├── Radmin VPN ── Piregram Network
User B ──┘
```
